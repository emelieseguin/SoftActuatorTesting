ADR 0005 outcome · 2026-07-11

Instrument Console is the normal shell.

Persistent operator state and an unmistakable Global Stop outweigh Experiment Studio's lower presenter-migration cost and stronger occasional-user guidance.

Selected direction, not production acceptance

02 · Confidence boundary

No human-operator study was available.

Do not turn inspection into invented evidence. The decision uses implementations, tests, documents, and reference images. Scores are not measured task completion or error rates. Representative operator feedback remains a mandatory pre-production evidence gap.

The repository contains no standalone approved-plan file. ADR 0005's pre-committed criteria and the tracked task graph are the concrete plan evidence.

03 · Rejected shell, retained ideas

Keep guidance—not a second application.

Readiness narrative

Show missing prerequisites and a recommended next action from presenter-owned state.

Progressive disclosure

Collapse advanced diagnostics so occasional users can parse the Console without losing expert depth.

Optional stage progress

Offer revisitable guidance inside the selected shell without creating another workflow state machine.

Light theme + safe defaults

Reuse light tokens and provide a recoverable default layout; theme and layout are not navigation architectures.

04 · Mandatory pre-production work

The current demo Stop is not production-ready.

  1. presenter-state-integration first. Replace demo environment, widget-label reads, local completion, and scenario flags with shared application snapshots and commands.
  2. One idempotent abort command. Specify starting/running/stopping/disconnected/duplicate/timeout/fault behavior and keep clean completion separate.
  3. Never hide safety chrome. Run/fault state and high-contrast Global Stop remain fixed, non-dockable, keyboard-reachable, and presenter-driven.
  4. Bring Studio guidance into Console. Use the same readiness source; do not introduce another state model.
  5. Harden layout persistence. Version, validate, restore transactionally, and fall back to an on-screen safe default without device access.
  6. Complete missing evidence. Windows/Linux, 1280×720 and 1920×1080, 100/150/200% scaling, screen reader, tab order, contrast, target size, and representative operators.
